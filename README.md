# 🎤 GreenMic

**キーを押している間だけ録音 → 離した瞬間に自動入力。声でサクサク文字を打てる Windows 用音声入力サポートツールです。**

[![Ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/esonobus)

---

## ✨ 主な機能

- 🎙️ **ホットキー録音** — キーを押している間だけ録音（デフォルト: 右 Ctrl キー）
- 📋 **自動テキスト入力** — 認識結果をクリップボード経由で自動貼り付け
- 📝 **入力プレビュー** — 入力後に通知バーへ認識テキストを約 1.5 秒表示
- ⚙️ **設定ウィンドウ** — トレイアイコン右クリック → 設定 から GUI で変更可能
  - ホットキーのカスタマイズ（お好みのキーに変更可能）
  - 認識言語の選択（7 言語対応）
  - 句読点の自動挿入 ON/OFF（日本語のみ）
- 💾 **設定の永続化** — `greenmic_config.json` に保存、次回起動時も維持
- 🌍 **多言語対応** — 日本語 / English / 中文 / 한국어 / Français / Deutsch / Español
- 🔒 **二重起動防止** — Windows Mutex による排他制御

## 📥 ダウンロード・インストール

[Releases](https://github.com/esono-bus/GreenMic/releases) から最新の `GreenMic_Setup.exe` をダウンロードしてインストールしてください。

> **注意**: 初回起動時に「Windows によって PC が保護されました」という画面が出る場合は、「詳細情報」→「実行」をクリックしてください。個人開発の無料ソフトウェアのため発行元が不明と表示されますが、安全です。

## 🚀 使い方

1. アプリを起動（デスクトップのアイコンをダブルクリック）
2. 文字を入力したい場所にカーソルを置く（メモ帳・Word・ブラウザなど）
3. **右 Ctrl キーを押しっぱなし** にして話す（「🎤 録音しています...」と表示）
4. 話し終わったら **キーを離す** → 自動で文字が入力される

---

## ⚙️ 設定の変更

タスクトレイの緑アイコンを **右クリック → 「設定」** で設定ウィンドウが開きます。

| 設定項目 | 説明 |
|---------|------|
| ホットキー | 録音に使うキーを変更できます（「変更」ボタンを押して任意のキーを押すだけ） |
| 認識言語 | 7 言語から選択できます |
| 句読点の自動挿入 | 日本語テキストに「。」「、」を自動付与します（日本語モード時のみ有効） |

## 📋 動作要件

- Windows 10 / 11
- インターネット接続（音声認識に使用）
- マイク（内蔵・外付け問わず）
